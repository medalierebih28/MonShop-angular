# 🛒 MonShop Angular

## 📌 Description
Application web de gestion de produits, clients et commandes développée avec Angular.

## 🚀 Fonctionnalités
- Authentification (login / signup)
- Dashboard
- Gestion des produits (CRUD)
- Navigation entre pages
- Interface utilisateur moderne

## 🛠️ Technologies
- Angular
- TypeScript
- HTML / CSS
